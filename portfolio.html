Canvas Games

Using Javascript for the game logic and Canvas to render, I have recreated several simple classic games.

Snake

Use the arrow keys to move your snake and eat apples, carefully managing your snake's size

Minesweeper

Using the math related clues, try to find all the bombs without setting any off!

Helicopter Game

Control your altitude with the mouse to avoid crashing into the ceiling, floor, and obstacles

Chess

The classic game Chess! Use strategy to capture your opponent's King! (No I did not make a chess engine lol)
